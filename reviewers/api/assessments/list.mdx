---
title: "List Assessments"
api: "GET /v1/assessments"
description: "List all assessments for your organization."
---

Returns all active (non-deleted) assessments for your organization, ordered by creation date. Each assessment includes linked OSS issue details if applicable.

### Authentication

Requires an org API key.

```
Authorization: Bearer psk_live_...
```

<ResponseField name="assessments" type="array">
  List of assessment objects, each with an optional `issue` field containing linked OSS issue details.
</ResponseField>
