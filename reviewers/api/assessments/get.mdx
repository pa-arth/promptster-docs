---
title: "Get Assessment"
api: "GET /v1/assessments/:id"
description: "Retrieve a single assessment by ID."
---

Returns the full assessment object including linked OSS issue details if applicable.

### Authentication

Requires either an org API key or a candidate key (for CLI access).

```
Authorization: Bearer psk_live_...
```

<ParamField path="id" type="string" required>
  The assessment ID (UUID).
</ParamField>

<ResponseField name="assessment" type="object">
  The assessment object with an `issue` field containing linked OSS issue details (or `null`).
</ResponseField>
