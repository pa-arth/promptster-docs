---
title: "Get Session Artifacts"
api: "GET /v1/sessions/:id/artifacts"
description: "Retrieve all derived artifacts for a session."
---

Returns all artifacts generated by the worker pipeline for a session. Artifacts include analytics, key moments, prompt analysis, test results, candidate summary, and more.

### Authentication

Requires either an org API key or a candidate key.

```
Authorization: Bearer psk_live_...
```

<ParamField path="id" type="string" required>
  The session ID.
</ParamField>

<ResponseField name="artifacts" type="array">
  List of artifact objects.
  <Expandable title="properties">
    <ResponseField name="id" type="string">Artifact ID (UUID).</ResponseField>
    <ResponseField name="sessionId" type="string">Parent session ID.</ResponseField>
    <ResponseField name="orgId" type="string">Organization ID.</ResponseField>
    <ResponseField name="kind" type="string">
      Artifact type. One of: `analytics_v1`, `data_points_v1`, `prompt_analysis_v1`, `tool_proficiency_v1`, `key_moments_v1`, `test_results_v1`, `candidate_summary`, `interview_followups_v1`, `codebase_tree_v1`, `candidate_files_v1`.
    </ResponseField>
    <ResponseField name="data" type="object">Artifact payload (structure varies by `kind`).</ResponseField>
    <ResponseField name="derivationV" type="integer">Derivation version number.</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>
