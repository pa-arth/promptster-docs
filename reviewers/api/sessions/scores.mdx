## Get Scores

<br />

```
GET /v1/sessions/:id/scores
```

Retrieves all reviewer rubric scores for a session, plus the computed Prompter Score if available.

## Submit Scores

<br />

```
POST /v1/sessions/:id/scores
```

Submit or update rubric scores for a session. Scores are upserted per dimension -- submitting a score for a dimension that already has one will update it.

### Authentication

Requires either an org API key or a candidate key.

```
Authorization: Bearer psk_live_...
```

<ParamField path="id" type="string" required>
  The session ID.
</ParamField>

### POST Body

The request body is an array of score objects (1 to 5 items).

<ParamField body="dimension" type="string" required>
  Score dimension. One of: `problem_decomposition`, `ai_orchestration`, `code_quality`, `debugging_process`, `communication`.
</ParamField>

<ParamField body="score" type="integer" required>
  Score value from 1 to 5.
</ParamField>

<ParamField body="notes" type="string">
  Optional reviewer notes for this dimension.
</ParamField>

### GET Response

<ResponseField name="scores" type="array">
  List of rubric score objects.
  <Expandable title="properties">
    <ResponseField name="id" type="string">Score record ID.</ResponseField>
    <ResponseField name="sessionId" type="string">Session ID.</ResponseField>
    <ResponseField name="dimension" type="string">Score dimension.</ResponseField>
    <ResponseField name="score" type="integer">Score value (1-5).</ResponseField>
    <ResponseField name="notes" type="string | null">Reviewer notes.</ResponseField>
    <ResponseField name="reviewerId" type="string | null">ID of the reviewer who submitted.</ResponseField>
    <ResponseField name="createdAt" type="string">ISO 8601 timestamp.</ResponseField>
    <ResponseField name="updatedAt" type="string">ISO 8601 timestamp.</ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="prompterScore" type="object | null">
  The computed Prompter Score artifact, or `null` if not yet computed.
</ResponseField>
